# X Lobster Hat Extension 🦞

这是一个 Chrome 插件，专门为 X (Twitter) 用户设计。

## 功能
- 检测推文中出现的 "openclaw" 关键词。
- 自动为推主头像戴上一顶龙虾帽。
- 提到次数越多，帽子颜色越深。

## 安装方法
1. 下载本仓库。
2. 打开 Chrome `chrome://extensions/`。
3. 开启“开发者模式”。
4. 加载已解压的扩展程序。
