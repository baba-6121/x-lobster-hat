# X Lobster Hat Extension 🦞✨

**让你的 X (Twitter) 时间线充满龙虾的气息！**

这是一个趣味性的 Chrome 插件，专门为 OpenClaw 社区爱好者设计。

## 🌟 核心功能
- **龙虾侦测**：自动检测推文中提到的 "openclaw" 关键词。
- **动态戴帽**：只要提到过关键词，该用户在你的时间线、个人主页、侧边栏的所有头像都会戴上一顶可爱的龙虾帽。
- **进化色彩**：提到次数越多，龙虾帽的颜色就越深（从粉红到深红）。
- **至尊之王**：实时统计“龙虾排行榜”，当前的 Top 1 用户将获得**金色皇冠**，且其头像上的龙虾帽会带有**金色发光特效**！
- **灵动动画**：帽子带有轻微的浮动和旋转动画，让你的时间线活起来。

## 🛠️ 技术亮点
- **Manifest V3** 标准。
- **精密追踪**：基于 `tweetId` 进行统计，防止滚动重复计分。
- **极致性能**：使用 `MutationObserver` + `Debounce` 防抖处理，确保页面滚动流畅。
- **全场景适配**：覆盖 Timeline, Profile, UserCell 等多种 X 页面布局。

## 📦 安装方法
1. 下载/克隆本仓库到本地。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择 `x-lobster-hat` 目录。
5. 刷新你的 https://x.com 页面！
