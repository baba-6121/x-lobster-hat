# 🦞 X Lobster Hat (X 龙虾帽)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-1.2.1-orange.svg)]()
[![Platform](https://img.shields.io/badge/platform-Chrome%20|%20Edge-blue.svg)]()

**Make your X (Twitter) timeline lobster-ific!**  
这是一个趣味性的 Chrome 扩展程序，专门为 X (Twitter) 用户和 OpenClaw 社区爱好者设计。只要有人提到特定关键词，就给他们戴上龙虾帽！

---

## 🌟 核心功能 (Features)

- **🦞 龙虾探测器**：自动识别推文中的关键词（支持 `openclaw` 和中文 `龙虾`）。
- **🎩 动态戴帽**：只要该用户提到过关键词，他在你的时间线、个人主页、侧边栏的所有头像都会戴上一顶可爱的龙虾帽。
- **📈 进化体系**：
  - **颜色加深**：提到次数越多，帽子颜色越红。
  - **传说级 (50+)**：开启蓝色赛博特效。
  - **神话级 (100+)**：开启紫色霓虹特效与脉冲光晕。
- **🏆 龙虾王排行榜**：弹窗实时显示谁是你关注圈里的“捕虾达人”，Top 1 拥有专属**金色皇冠**和**全站发光特效**。
- **🎨 极致自定义**：
  - 支持**上传自定义 SVG** 替换默认龙虾。
  - 自由调节帽子基础颜色。
  - 实时高亮推文中的关键词。
- **🌍 国际化支持**：中英双语界面，一键切换。

## 🚀 快速开始 (Quick Start)

### 安装方法
1.  **下载代码**：点击 `Code -> Download ZIP` 或使用 `git clone` 本仓库。
2.  **打开扩展程序页**：在 Chrome 地址栏输入 `chrome://extensions/`。
3.  **开启开发者模式**：勾选右上角的“开发者模式 (Developer mode)”。
4.  **加载插件**：点击“加载已解压的扩展程序 (Load unpacked)”，选择本项目文件夹。
5.  **刷新 X**：打开 [x.com](https://x.com)，开启你的捕虾之旅！

## 🛠️ 技术亮点 (Tech Specs)

- **Manifest V3**：遵循最新 Chrome 扩展标准。
- **零延迟渲染**：采用内存配置缓存机制，即使在快速滚动时也能平滑加载。
- **精准统计**：基于 `tweetId` 进行唯一性校验，彻底杜绝重复计分。
- **DOM 韧性**：递归式父容器扫描逻辑，确保帽子在任何复杂的布局下都不会被遮挡。
